# Налаштування Telegram-боту для заявок

## 1. Створіть бота в Telegram

1. Відкрийте [@BotFather](https://t.me/BotFather) в Telegram
2. Відправте `/newbot` і вкажіть ім'я бота
3. Скопіюйте **токен** (виглядає як `5192442579:AAHxxxxxxxxxxxx`)

## 2. Отримайте Chat ID

1. Напишіть вашому боту будь-яке повідомлення (наприклад, /start)
2. Відкрийте в браузері:  
   `https://api.telegram.org/bot<ВАШ_ТОКЕН>/getUpdates`
3. Знайдіть `"chat":{"id":XXXXXXXXX}` — це ваш Chat ID  
   (якщо у вас 5192442579 — можна використати його)

## 3. Задеплойте на Vercel

1. Зареєструйтесь на [vercel.com](https://vercel.com)
2. Імпортуйте проєкт (папку з сайтом)
3. В налаштуваннях проєкту → **Settings → Environment Variables** додайте:
   - `TELEGRAM_BOT_TOKEN` = ваш токен з BotFather
   - `TELEGRAM_CHAT_ID` = 5192442579 (або ваш Chat ID)

4. Зробіть redeploy

## 4. Результат

Заявки з форм «Замовити дзвінок» та «Надіслати заявку» надходитимуть у ваш Telegram.  
Клієнт побачить: **«Заявку відправлено! Ми вам передзвонимо ✅»**
